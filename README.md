# Blog app using Ruby on Rails

This app is a simple blog app that a user can create, read, update, and delete articles and users entries.

* Ruby version
```
$ ruby 3.1.2
$ Rails 7.0.3.1
```

* Setup
```
 $ bundle install
```

* Database creation
```
 $ rake db:setup
```

* How to run the test suite
```
$ rails test
```

* Starting the application
```
$ rails server
```
